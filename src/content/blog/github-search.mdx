Github的搜索功能我想大家应该都使用过。我现在面临的场景是这样的：查找tauri项目，但是一些star比较多的我都看过了，或者项目已经比较大了，不适合我这种新手去入门，所以我想搜索一些star不是很多tauri项目。

### 搜索语法

- **>n**: 大于
- **\<n**: 小于
- **>=n**, **n..**: 大于等于
- **\<=n**, **..n**: 小于等于
- **m..n**: 大于m小于n
- **NOT keyword**: 排除
- **-filter**: 排除

上面这些操作符搭配可以搜索的项使用

- **stars**: star数量
- **created**: 创建时间
- **pushed**: 更新时间
- **language**: 编程语言
- **topic**: 主题
- **repo**: 仓库

- `topic:tauri stars:0..100`:
