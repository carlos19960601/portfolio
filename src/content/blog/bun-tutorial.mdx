## `bun run` vs `bunx` 对比有什么不同？

| 特性                          | `bun run`                        | `bunx`                                              |
| ----------------------------- | -------------------------------- | --------------------------------------------------- |
| **是否自动安装包**            | ❌ 不会安装，必须已存在          | ✅ 如果本地没有，会自动从 npm 下载并缓存            |
| **读取 `package.json` 脚本**  | ✅ 是（如 `bun run dev`）        | ❌ 否                                               |
| **运行本地 `.ts`/`.js` 文件** | ✅ 支持（如 `bun run index.ts`） | ❌ 不支持（只运行 CLI 可执行文件）                  |
| **典型用途**                  | 项目开发脚本、运行本地代码       | 一次性使用 CLI 工具（如 `create-vite`, `prettier`） |
| **类似 Node.js 命令**         | `npm run` + `node`               | `npx`                                               |
| **性能**                      | 极快（直接运行）                 | 快（首次下载后缓存，后续秒开）                      |
